# 🌸 Iris Dataset - Exploratory Data Analysis (EDA)

## 📊 Project Overview
This project performs a comprehensive Exploratory Data Analysis (EDA) on the famous **Iris Dataset** collected by British statistician Ronald Fisher in 1936.

## 📁 Dataset Information
- **Total Samples:** 150 rows
- **Features:** 4 numerical (sepal length, sepal width, petal length, petal width)
- **Target Classes:** 3 (Setosa, Versicolor, Virginica) - 50 samples each
- **Missing Values:** None - perfectly clean dataset

## 🎯 Project Objectives
- Load and inspect the Iris dataset
- Perform statistical analysis
- Create visualizations to understand data patterns
- Identify relationships between features
- Determine which features best separate species
- Generate comprehensive report with findings

## 📈 Key Findings
1. **Perfectly Balanced Dataset:** 50 samples per species
2. **Petal features** are the most discriminative for species classification
3. **Setosa** is completely linearly separable from other species
4. **Petal length and width** are highly correlated (r = 0.96) - feature redundancy
5. **Sepal width** is the least informative feature
